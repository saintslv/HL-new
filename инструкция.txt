Положить папку Volumes в диск С:

Выполнить следующие команды в cmd:
1. docker-compose build
2. docker-compose up -d

После этого все сервисы должны быть запущены

Затем использовать запрос /posts/feed из коллекции postman. 
При выполнении запроса приложение проверит наличие кэша в Redis.
Если кэш есть, то данные для ответа вернутся из кэша.
Если кэша нет, то приложение получит данные из postgres, создаст кэш в Redis и вернет ответ.
Инвалидация кэша произойдет через час после создания.
